Bootstrap

Bootstrap is a free collection of tools for creating websites and web applications. It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions.It is very popular tool and it has been used by NASA and MSNBC, among many others.

History

  • Developed by Mark Otto and Jacob Thornton at Twitter in mid-2010 as a framework to encourage consistency across internal tools.
  • In February 2012, it was the most popular GitHub development project.
  • First public 1.0 release was on Friday, August 19, 2011, and it was also the day it was open sourced

Features

  • Bootstrap is compatible with the latest versions of all major browsers
  • Gracefully degrades when used on older browsers such as Internet Explorer 8
  • Since version 2.0 it also supports responsive design

Some Advantages of using Bootstrap

  • It saves a lot of time
  • Customizable
  • Lots of available plugins
  • Easy to learn and has a very good documentation
  • Has a huge active community
